============
ShipWeightOS
============

Description
===========

ShipWeightOS is an open-source weights tracking program for water-borne vessels.

References
==========

1)  NSTM Chapter 096 - Weights and Stability (NAVSEA S9086-C6-STM-010/CH-096R1).
	Naval Sea Systems Command, 2 AUG 1996.
